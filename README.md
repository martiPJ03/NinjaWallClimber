# 🥷 Ninja Wall Runner (Working Title)

A 2D vertical endless runner hyper-casual mobile game for Android, developed using Unity and C#. This project is designed under strict scope constraints, targeting a 2-month development cycle at approximately 1 hour per day.

## 🚀 Game Features
* **Core Mechanic:** One-touch controls. The ninja automatically slides down walls and jumps to the opposite side to dodge obstacles when the screen is tapped.
* **Visual Style:** Minimalist 2D Pixel Art optimized for mobile performance.
* **Procedural Generation:** Endless vertical scrolling with dynamic obstacle and collectible spawning.
* **Monetization:** Free-to-Play model utilizing Unity Ads / AdMob (Rewarded Ads for revives and Interstitials between runs).

## 🛠️ Tech Stack
* **Engine:** Unity 2D
* **Language:** C#
* **Target Platform:** Android (API 29+)
* **Assets:** Custom pixel art combined with open-source assets (e.g., Kenney.nl).

## 📅 Development Roadmap (60 Hours)
1. **Phase 1: Core Prototype** – Wall jump mechanics, physics tuning, and collision detection. *(Completed)*
2. **Phase 2: Game Loop** – Score system, endless vertical scrolling, and procedural obstacle spawning.
3. **Phase 3: UI & Economy** – Main menus, Game Over screens, local high-score saving, and a basic skin shop.
4. **Phase 4: Game Juice & Polish** – Sound effects, pixel particles, and visual feedback.
5. **Phase 5: Monetization & Release** – Ad SDK integration, Android hardware testing, and Google Play Store publishing.

## 💻 How to Test the Prototype
1. Clone this repository.
2. Open the project in Unity.
3. Set the **Game View** aspect ratio to **Portrait (9:16)**.
4. Press *Play* and use **Left Mouse Click** to simulate mobile screen taps.
